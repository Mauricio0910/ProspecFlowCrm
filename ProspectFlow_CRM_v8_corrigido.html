Configure uma vez — toda a equipe compartilha os dados em tempo real
Passo 1 — Criar projeto no Firebase
1. Acesse console.firebase.google.com
2. Clique em Adicionar projeto → dê um nome (ex: "prospectflow-crm")
3. Desative o Google Analytics (opcional) → Criar projeto
4. No menu lateral, vá em Firestore Database
5. Clique em Criar banco de dados → Modo de teste → Próximo → Concluir
6. Ainda no menu, vá em Configurações do projeto (ícone ⚙) → aba Geral
7. Role para baixo → clique em </> Web → registre o app → copie o objeto firebaseConfig
Abrir Console Firebase ↗
Passo 2 — Cole sua configuração Firebase
Cole todo o bloco do firebaseConfig — pode incluir o const firebaseConfig = no início, sem problema:
// Exemplo do que colar: { apiKey: "AIzaSy...", authDomain: "meu-projeto.firebaseapp.com", projectId: "meu-projeto", storageBucket: "meu-projeto.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123" }
